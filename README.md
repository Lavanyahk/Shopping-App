# Shopping App

A multi-page shopping application built using HTML, CSS, and JavaScript.

## Pages
- Home Page
- Login Page
- Register Page
- Shop Page

## Features
- User-friendly UI
- Login and registration pages
- Product listing in shop page
- JavaScript-based shopping logic

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
Open home.html in a web browser and navigate through the app.
